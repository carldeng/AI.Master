## 顾问介绍
- 目标达成顾问：[查看使用方式](./GoalMaster/README_zh.md)
    - 把一句话目标，变为靠谱，又有一定挑战的目标
    - 面对不同的目标，采用不同达成策略
    - 帮助你制定明确的目标实施计划
    - 帮助你建立目标反馈和调整机制
    - 教你如何应对挫折和失败
    - 关注你的个人特性和内在的需求
    - 提供持续的支持和鼓励

## 项目特色
- 像真正的顾问一样提供专业的服务
- 使用成熟的方法论来生成内容
- 无需记忆复杂指令，可像自然语言一样对话
- 同时掌握多种技能，可根据不同的需求，切换不同的技能
- 根据个人特性定制化内容
- 可直接在chatGPT中使用
